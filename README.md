## About This Page
**KPMG – Jenkins and Okta Integration (Proof of Concept)**

This Proof of Concept (POC) demonstrates the integration of **Jenkins** with **Okta** to establish Okta as the **Identity Provider (IdP)** for Jenkins, enabling secure **Single Sign-On (SSO)** using the **SAML 2.0** protocol.

In addition to SSO integration, the project implements a **CI/CD pipeline** that automates the **onboarding of standard pattern applications into Okta**, streamlining identity lifecycle management and reducing manual configuration effort.

### Key Objectives

- Configure Okta as the IdP for Jenkins using SAML-based SSO
- Integrate Jenkins with Okta for authentication and user provisioning
- Develop a CI/CD pipeline to automate standard application onboarding into Okta
- Demonstrate improved efficiency, consistency, and security in the application onboarding process

### Technologies & Tools

- Okta – Identity and Access Management
- Jenkins – CI/CD automation server
- SAML 2.0 – Authentication and SSO protocol
- Groovy / Shell scripts – Pipeline automation
- GitHub – Version control and source management
